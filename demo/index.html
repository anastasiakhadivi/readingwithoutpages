CSS Animations

A css animation makes elements change from one style to another over time.
In order to use css animations, you have to set keyframes. 

				
	#box1{
		animation-name: animationName1;
		animation-duration: 1s;
		animation-iteration-count: 5;
		margin-left: auto;
		margin-right: auto;
		margin-top: 10%;
	}

	@keyframes animationName1{
		0%{
			width: 50px;
			height: 50px;
			background-color: yellow;
			opacity: 1;
		}
		100%{
			width: 80px;
			height: 80px;
			opacity: 0;
		}
	}
				
			
				
	#box2{
		animation-name: animationName2;
		animation-duration: 8s;
		animation-iteration-count: infinite;
	}

	@keyframes animationName2{
	0%{
		width: 50px;
		height: 50px;
		background-color: blue;
		opacity: .2
	}
	10%{
		background-color: yellow;
		opacity: 1;
	}

	35%{
		height: 30%;
		background-color: #0f34c6;
		border: 5px yellow dotted;
	}
	100%{
		background-color: pink;
		height: 100%;
		opacity: .1
		width:100%;
	}
}
				
		
				
	p{
		animation-name: animationName3;
		animation-duration: 4s;
		animation-direction: alternate-reverse;
		animation-iteration-count: infinite;
		font-size: 3em
	}

	@keyframes animationName3{
		0%{
			font-size: 3em;
		}
		100%{
			font-size: 0em;
		}	
	}

You can target anything!

				
	p{
		animation-name: animationName3;
		animation-duration: 4s;
		animation-direction: alternate-reverse;
		animation-iteration-count: infinite;
	}

	@keyframes animationName4{
		0%{
			background-color: yellow;
		}
		100%{
			margin-left: 35%;
		}
	}
